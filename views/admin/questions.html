Nhập câu hỏi từ file CSV
🚀 Mới: Hỗ trợ file Excel trực tiếp và 3 cột!

1. Cách dễ nhất - File Excel (.xlsx, .xls):
• Cột A: Câu hỏi
• Cột B: Câu trả lời
• Cột C: Danh mục (Khởi Động / Vượt Chướng Ngại Vật / Tăng Tốc / Về Đích)
• Upload trực tiếp, không cần export ✅

2. CSV/TXT với Header (khuyến nghị):
• Dòng 1: Question,Answer,Category hoặc Qus,Ans,Danh mục
• Từ dòng 2: Dữ liệu câu hỏi, đáp án và danh mục

3. CSV/TXT không Header (3 cột):
• Mỗi dòng: Câu hỏi,Đáp án,Danh mục
• Nếu không có cột 3, mặc định sẽ là "Khởi Động"

4. Formats hỗ trợ:
• Excel: .xlsx, .xls ⭐ (Khuyến nghị)
• CSV: .csv
• Text: .txt (tab-separated)

💡 Với file Excel, bạn có thể dùng dấu phẩy, xuống dòng tự do!
